## 你好 👋

欢迎来到我的个人网站。

我是 AI 全栈开发工程师，专注于 **LLM 应用落地** 与 **全栈开发**。从大模型的 API 集成到前端交互界面，从后端服务架构到部署运维，我喜欢端到端地解决问题。

### 为什么做这个网站？

过去几年，AI 领域的变化太快了。几乎每个月都有新的模型、新的框架、新的最佳实践出现。这个博客是我记录和整理这些信息的地方——一方面是给自己做知识沉淀，另一方面也希望给正在走同样路的人一些参考。

这里不会有那种"AI 会取代程序员"的宏大叙事，更多的是：

- **实战经验**：某个功能怎么实现的，踩了什么坑
- **技术选型**：为什么选 A 而不是 B，实际用下来怎么样
- **项目复盘**：从 idea 到上线，过程中做了什么决策

### 关于技术方向

目前的核心关注点：

- **LLM 应用架构**：RAG、Agent、Function Calling 的工程化落地
- **前端 & 全栈**：React、Next.js、Tailwind CSS 的实践
- **开发工具**：AI 辅助编程、Docker 部署、CI/CD

### 写在最后

这个站点本身也是我的实验场——你会看到它随着时间不断进化。每一行代码、每一篇文字，都是对"学以致用"这四个字的践行。

欢迎通过邮件或 GitHub 与我交流。
